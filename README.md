# Admin Panel Selenium TestNG Framework

## Project Overview
This project is a Selenium Java Automation Framework developed using:
- Selenium WebDriver
- Java
- TestNG
- Maven
- ExtentReports

## Features
- Form validation testing
- Table operations testing
- Alert handling
- Dynamic element handling
- Frame handling
- Screenshot capture on failure
- HTML report generation

## Framework Design
- Page Object Model (POM)
- Reusable utilities
- Config-driven framework
- TestNG suite execution

## Tools & Technologies
- Java
- Selenium
- TestNG
- Maven
- WebDriverManager
- ExtentReports

## Run Command

```bash
mvn test
```
